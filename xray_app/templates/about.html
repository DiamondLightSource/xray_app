This website is built arount xraylib, an ANSI-C library that provides access to compiled physical data on xray-matter interactions. The library comes with bindings to support lots of languages including such as Python, Perl, Fortran, PHP and others. For information on the library itself, have a look at the GitHub repository.

This interface will allow you to make simple queries as well as plot graphs of cross section data.
